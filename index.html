Objective

Build a predictive model using stock metrics, news sentiment, and economic indicators. Implement NEAT-enhanced DDPG to adaptively forecast trends and make trading decisions.

Problem Statement

Forecast stock market trends using reinforcement learning and evolutionary algorithms with multimodal inputs like news sentiment and financial data.

Proposed Architecture

Uses NEAT for dynamic policy evolution over DDPG.

Methodology

  • Feature engineering with stock indicators
  • Optimized reward function
  • Integrated NEAT with DDPG
  • Evaluated using Sharpe Ratio, Max Drawdown, etc.

Reward Function

  • Maximizes profit while considering risk, cost, and time value
  • Sharpe ratio used for risk-adjusted reward
  • Transaction cost and market realism included

Results

  • Higher Sharpe Ratio with NEAT-DDPG
  • Reduced drawdown and volatility
  • Outperforms Buy-and-Hold strategy

Conclusion

NEAT-DDPG provides a stable, adaptive, and profitable strategy for stock trend prediction using RL with real-world constraints.
